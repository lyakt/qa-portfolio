## Bug Reports
### Crushing Blow does not break objects instantly
- Platform: PC (Windows)
- Type: Gameplay bug
- Reproducibility: Always

**Steps to reproduce:**
1. Launch Fracture Field Demo
2. Start a new game
3. Unlock Crushing Blow ability
4. Use it on breakable objects

**Expected result:**
Crushing Blow breaks objects instantly.

**Actual result:**
Objects break only after 3–4 attempts.
