# How Many Dudes Demo — QA Feedback

## Positive
- Very good graphics quality for an indie game.
- The player can upgrade the character and freely experiment with gameplay.
- The game includes different types of entities such as animals (frogs, geese, etc.).
- There is an option to add zombies, which adds variety.
- The core idea of the game is interesting and combines strategy, simulator, and indie genres well.
- The game is available in multiple languages, which is a big plus for accessibility.
